🚀 AI Resume Analyzer  <br>

A simple AI-powered web app built using the MERN stack that analyzes resumes and provides useful insights like skill extraction, scoring, and improvement suggestions.

✨ Features
Upload resumes (PDF/DOC)  <br>
AI-based resume analysis  <br>
Skill extraction & keyword detection  <br>
Resume scoring (ATS-based)  <br>

Suggestions to improve resume
🛠️ Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Other: AI API, Multer

⚙️ How to Run
git clone <your-repo-link> <br>
cd ai-resume-analyzer  <br>
npm install  <br>
npm run dev  <br>

📌 About Project
This project is built to help users improve their resumes by analyzing content and suggesting better structure, keywords, and skills required for job roles.
